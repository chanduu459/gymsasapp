# Project Root

## Structure

- `frontend/` → React + Vite + TypeScript app
- `backend/` → Node.js + Express + PostgreSQL API

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

Backend default: `http://localhost:3001`
Frontend default: `http://localhost:5173`
